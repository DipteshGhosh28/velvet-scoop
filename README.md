# Velvet Scoop

  An ultra-luxury ice cream brand website — dark Parisian aesthetic with cinematic animations.

  ## Tech Stack

  - **React + Vite** (TypeScript)
  - **Framer Motion** — animations and transitions
  - **Tailwind CSS** — styling
  - **Lucide React** — icons

  ## Features

  - Cinematic preloader with real asset-progress tracking
  - Custom gold cursor (desktop)
  - Full-screen hero video with poster fallback
  - Flavor collection with photo-to-video crossfade on hover / tap
  - Animated stats counter
  - Fully responsive (mobile, tablet, desktop)
  - Hamburger mobile menu
  - Reservation form with custom LuxurySelect dropdowns
  - Confirmation screen after submission
  - Smooth scroll with Framer Motion in-view animations

  ## Color Palette
